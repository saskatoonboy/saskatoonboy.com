There's a problem though this method doesn't work for one big reason. Let me give you an example. 3 + (-3) = 0, right? When you add a negative number that's the same as subtracting the postive. The problem is if we do 0011 + 1011 = 1011. 1011 is -5! That's not right so this systeme works to represent the numbers but as soon as you start doing math it breaks.

Two's complement

This next method is almost the same as 1's complement and it is called 2's complement. In 2's complement you follow the same steps as 1's complement but you have one extra step at the end. The extra step is simply to add one. So if we have 3 which is 00011 then we flip the digits to get -3 which is 11100 and finally we add one to get 11101.
